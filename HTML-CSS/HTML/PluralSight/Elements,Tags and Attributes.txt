Elements,Tags and Attributes

Element: Different commands within the HTML file
Tag: Name of the HTML Element
Attribute: Additional parameter or setting that specifies information for greater customization


An Element has a Tag which is the exact name of the 

ie. <head>, <title>, <body>, <h1>, <p>

Elements can inlcude attributes like src, title, href

ie. <img src="box.png" title="Blue Box">




The most common attributes are the id and class attributes

ie.     <img id="box" class="shapes">

id= Gives a unique name to elements on the page
class= gives shape to the element on the page
