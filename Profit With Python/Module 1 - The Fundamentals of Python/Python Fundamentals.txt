#Division
If you're dividing decimals (floats) you can use a double slash '//' to round the division to a whole number
Ex.

9.5 / 2 = 4.75
9.5 // 2 = 4.0



#Modular Division
Divide numbers and provides the remainder as the answer
Ex.

9 % 3 = 0
10 % 3 = 1
